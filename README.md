# 豆绘 Perler Studio

一个拼豆图纸 Web App，支持图片转图纸、自由创作和服务端账号系统。

在线使用：<https://eric-zhouxi.github.io/perler-bead-studio/>

## 功能

- 可切换 MARD 221 / 291 标准色卡，并使用 Lab 色差转换图片
- 提供 20 个常用色快捷选择，完整色板仍保留相同颜色
- 图片等比例居中，画布多余区域保持空白
- 优先识别轮廓线，再识别轮廓内颜色
- 同一图片生成原始识别、净色优化、深色增强三个版本供切换选择
- 净色版本合并低必要性的孤立杂色，深色版本减少暗部偏灰和过渡中间色
- 页面背景使用局部鼠标字符水波、增强轨迹拖尾和惯性消散效果，鼠标点击与触屏点击会产生双层扩散波纹，并适配高分屏和减少动态效果设置
- 图片转换完成后默认锁定图纸，点击“开始编辑”后才允许修改
- 16×16、24×24、32×32、40×40、50×50 预设以及自定义规格
- 常见拼豆色板，点击颜色后直接在画布绘制
- 网格显示、无色号预览、缩放、撤销、清空与 PNG 导出
- 手机号支持密码、短信验证码登录和新用户注册
- 微信开放平台与 QQ 互联扫码登录
- 服务端保存头像、昵称、地区、历史图纸和 291 色库存
- 管理员可查看所有用户的图纸并审计删除操作

## 本地使用

图纸编辑功能可直接打开 `index.html` 使用。账号功能需要先部署 `server/`，再将 `api-config.js` 中的 `DOUHUI_API_BASE` 设置为 HTTPS API 地址。

推送到 `main` 分支后，GitHub Actions 会自动发布 GitHub Pages 网站。

后端配置、迁移、安全设置和腾讯云上线步骤见 `server/README.md`。管理员凭据只通过腾讯云环境变量或密钥管理服务注入，禁止写入前端、GitHub Actions 日志或仓库文件。

## 色卡参考

221 色卡使用 MARD 的 A、B、C、D、E、F、G、H、M 系列；291 色卡在此基础上增加 P、Q、R、T、Y、ZG 系列。屏幕 HEX 仅用于颜色匹配参考，实体拼豆会受批次、光线和显示器影响。

- <https://www.doudougongfang.com/kb/beads/mard-palette>
- <https://www.pindou.online/colors>
